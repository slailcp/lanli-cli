# 酒店PC版
正式线： 
测试线： https://hoteloppctest.fanjiaxing.com

## 安装依赖
```
npm install/cnpm i
```

### 本地运行
```
npm run serve/npm start
```

### env文件
```
NODE_ENV 环境
VUE_APP_BASE_URL 接口地址，可以在项目中直接取值 process.env.VUE_APP_BASE_URL
```

### 打包项目
```
测试线打包： npm run Debug
正式线打包： npm run Release
预发布一： npm run Test01
预发布二： npm run Test02

```
