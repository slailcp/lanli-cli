## 安装方式
```js
npm i -g lanli-cli
```
## 创建项目
```js
lanli-cli create projectname  // 默认从github上拉取
lanli-cli create projectname -g   // 从github上拉取
lanli-cli create projectname -l // 从缓存的模板中拉取
lanli-cli create projectname -n=E:\MyProject\vite  // 从本机E:\MyProject\vite文件下拉取
```

## 命令
```js
lanli-cli -V // 查看版本号
lanli-cli -h // 查看命令说明
```
